# Quickstart

## Rodar

```bash
npm install
npm run dev
```

Acessar: `http://localhost:3000`

## Validar

```bash
npm test
npm run build
```

## Cenários manuais

1. Abrir a página e validar hero/data/local/CTAs.
2. Enviar RSVP com nome e telefone.
3. Clicar em copiar Pix.
4. Validar layout em desktop e mobile.
