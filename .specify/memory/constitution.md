# Constituição do Projeto

## Princípios

1. **Valor para convidados primeiro**: a experiência deve ser clara, acolhedora e acessível para familiares em qualquer dispositivo.
2. **Privacidade e segurança**: dados de RSVP devem ser tratados como informação sensível; não expor dados reais em código ou prints.
3. **Acessibilidade**: contraste, navegação por teclado, textos alternativos e formulários com labels são obrigatórios.
4. **Performance e SEO**: site convite deve carregar rápido, ter metadados sociais e layout responsivo.
5. **Design emocional sem excesso**: visual infantil, moderno e delicado, evitando poluição visual.
6. **TDD/validação**: toda entrega precisa de build e validação visual real.
